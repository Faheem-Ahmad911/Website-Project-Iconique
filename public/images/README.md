# Image Placeholder Guide for TheIconique Website

This folder should contain all images for the homepage. Below are the required images and their specifications:

## Required Images

### 1. Logo
- **File**: `logo.png` or use the provided `logo.svg`
- **Recommended Size**: 200x50px (or higher resolution maintaining aspect ratio)
- **Format**: PNG with transparent background or SVG
- **Description**: TheIconique logo

### 2. Hero Section Images (Auto-scrolling carousel)
- **Files**: `hero1.jpg`, `hero2.jpg`, `hero3.jpg`, `hero4.jpg`
- **Recommended Size**: 1920x600px
- **Format**: JPG or PNG
- **Description**: High-quality cosmetic product images or lifestyle shots

### 3. Featured Product Images

- **Recommended Size**: 600x600px (square format)

## Image Optimization Tips

1. **Compress images** before uploading to reduce page load time
2. Use **WebP format** for better compression (with JPG fallback)
3. Maintain **aspect ratios** specified above for best display
4. Use **high-quality** images that represent your brand well
5. Ensure images have **good lighting** and clear product visibility

## Temporary Placeholders

Until you add real images, you can use placeholder services:

### Option 1: Via URL (requires internet)
Replace image sources in HTML with:
```html
<!-- For products (600x600) -->
<img src="https://via.placeholder.com/600x600/ffb6c1/ffffff?text=Product+Name" alt="Product">

<!-- For hero (1920x600) -->
<img src="https://via.placeholder.com/1920x600/ffc0cb/ffffff?text=Hero+Image" alt="Hero">

<!-- For bundles (800x600) -->
<img src="https://via.placeholder.com/800x600/ffb6c1/ffffff?text=Bundle+Name" alt="Bundle">
```

### Option 2: Create solid color placeholders
Use any image editing software to create solid pink/white gradient backgrounds with product names as text.

## Where to Find Images

1. **Stock Photo Websites** (for placeholders):
   - Unsplash.com (free)
   - Pexels.com (free)
   - Pixabay.com (free)

2. **Professional Product Photography**:
   - Hire a professional photographer
   - Use your actual product photos
   - Create lifestyle shots with models

3. **Search Keywords**: "cosmetics product photography", "makeup flatlay", "beauty products", "lipstick photography", etc.

## Important Notes

- Ensure you have **rights to use** any images you add
- Keep **consistent styling** across all product images
- Use **white or light backgrounds** for product images to match the site design
- Test images on both **mobile and desktop** views
- Update `logo.png` path in HTML if using a different format
